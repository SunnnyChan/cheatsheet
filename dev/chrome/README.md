# 截屏
```text
1. Ctrl+Shift+I（Windows）/F12 或者 Alt+Command+I（Mac）
2. Ctrl+Shift +P(Windows) 或 Command+Shift+P(Mac)
3. 输入"Screen"
  ◆ 整个网页：Capture full size screenshot
  ◆ 节点网页：Capture node screenshot
  ◆ 当前屏幕：Capture screenshot
```
